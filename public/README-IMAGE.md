# Как добавить изображение приветствия

## Инструкция:

1. **Сохраните ваше изображение** (постер Happy New Year) с именем `welcome-poster.jpg`

2. **Поместите файл** в папку `public/` вашего проекта:
   ```
   ny-sets-miniapp/
   └── public/
       └── welcome-poster.jpg  ← Сюда
   ```

3. Изображение автоматически появится на экране приветствия!

## Альтернативные форматы:

Если у вас изображение в другом формате (PNG, WEBP), можно:

1. Переименовать файл в `welcome-poster.jpg`
2. Или изменить путь в файле `src/components/WelcomeScreen.jsx`:
   ```jsx
   src="/welcome-poster.png"  // Измените расширение
   ```

## Размер изображения:

Рекомендуемые параметры:
- **Формат:** JPG, PNG, WEBP
- **Соотношение сторон:** Вертикальное (как постер)
- **Размер файла:** До 2MB для быстрой загрузки
- **Разрешение:** 1080x1920px или выше

## Проверка:

После добавления изображения:
1. Откройте http://localhost:5177/
2. Вы должны увидеть ваш постер на экране приветствия
3. Нажмите на экран → откроется форма заказа
