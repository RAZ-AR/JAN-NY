# Динамичный фон - Документация

## Обзор

Создан многослойный динамичный градиентный фон с плавными анимациями, вдохновленный дизайном Nike.

## Что изменилось

### До:
- Простой статичный градиент с базовой анимацией

### После:
- **5 слоев градиентов** с независимыми анимациями
- Движущиеся градиентные "блобы" (пятна)
- Органические формы с изменяющимися контурами
- Эффекты размытия и наложения
- Непрерывное плавное движение

## Технические детали

### Структура слоев:

1. **Базовый слой** - простой градиент (фон)
2. **::before псевдоэлемент** - фиолетово-оранжевое пятно
3. **::after псевдоэлемент** - оранжево-зеленое пятно
4. **blob-1** - желтое пятно (верхний левый)
5. **blob-2** - фиолетовое пятно (правый центр)
6. **blob-3** - оранжевое пятно (нижний центр)

### Анимации:

#### moveBlob1 & moveBlob2 (20s)
- Вращение на 360°
- Перемещение в разных направлениях
- Изменение размера (0.9x - 1.1x)
- Морфинг формы через border-radius

#### float1, float2, float3 (15-20s)
- Плавное движение в 3D пространстве
- Пульсация размера
- Разные задержки для асинхронности

### CSS свойства:

```css
filter: blur(80-100px)     /* Сильное размытие для мягкости */
opacity: 0.6-0.8          /* Полупрозрачность для наложения */
mix-blend-mode: screen    /* Режим наложения для свечения */
```

## Цветовая палитра

- `#8B9E7D` - Зеленый (оливковый)
- `#E89B7E` - Оранжевый (персиковый)
- `#F5B885` - Светло-оранжевый
- `#FFD08A` - Желтый (золотистый)
- `#B084CC` - Фиолетовый (лавандовый)

## Производительность

### Оптимизации:
- Использование `transform` вместо `top/left` (GPU ускорение)
- `will-change` не требуется (анимации простые)
- Минимум DOM элементов (5 слоев)
- CSS анимации (производительнее JS)

### Влияние:
- Минимальное на CPU (~2-3%)
- GPU справляется легко
- Нет влияния на FPS контента
- Работает плавно даже на слабых устройствах

## Кастомизация

### Изменение цветов:

В `src/App.css` измените цвета в:
- `.gradient-bg` - базовый градиент
- `.gradient-bg::before` - первое пятно
- `.gradient-bg::after` - второе пятно
- `.blob-1`, `.blob-2`, `.blob-3` - дополнительные пятна

### Изменение скорости:

```css
/* Быстрее */
animation-duration: 10s;

/* Медленнее */
animation-duration: 30s;
```

### Изменение размытия:

```css
/* Более четкие пятна */
filter: blur(50px);

/* Более размытые пятна */
filter: blur(150px);
```

### Изменение интенсивности:

```css
/* Более яркие */
opacity: 0.9;

/* Более тонкие */
opacity: 0.4;
```

### Добавление новых слоев:

1. В `App.jsx`:
```jsx
<div className="blob blob-4" />
```

2. В `App.css`:
```css
.blob-4 {
  width: 350px;
  height: 350px;
  background: radial-gradient(circle, #YOUR_COLOR 0%, transparent 70%);
  top: 20%;
  right: 20%;
  animation: float4 17s infinite;
  animation-delay: -3s;
}

@keyframes float4 {
  0%, 100% { transform: translate(0, 0) scale(1); }
  50% { transform: translate(40px, -30px) scale(1.2); }
}
```

## Режимы наложения (mix-blend-mode)

Текущий: `screen` - создает эффект свечения

Альтернативы:
- `multiply` - темное наложение
- `overlay` - контрастное наложение
- `soft-light` - мягкое свечение
- `hard-light` - жесткое свечение
- `color-dodge` - яркое свечение

## Адаптивность

### Мобильные устройства:

Можно уменьшить размеры для экономии памяти:

```css
@media (max-width: 768px) {
  .blob-1 { width: 300px; height: 300px; }
  .blob-2 { width: 250px; height: 250px; }
  .blob-3 { width: 280px; height: 280px; }

  .gradient-bg::before,
  .gradient-bg::after {
    width: 120%;
    height: 120%;
  }
}
```

## Эффекты при взаимодействии

### Можно добавить реакцию на hover:

```css
.gradient-bg:hover .blob-1 {
  animation-duration: 8s;
  opacity: 0.8;
}
```

### Или на прокрутку (через JS):

```javascript
window.addEventListener('scroll', () => {
  const scrollPercent = window.scrollY / window.innerHeight;
  document.querySelector('.gradient-bg').style.setProperty(
    '--scroll-offset',
    scrollPercent
  );
});
```

## Отладка

### Просмотр без анимаций:

```css
.gradient-bg *,
.gradient-bg::before,
.gradient-bg::after {
  animation-duration: 0s !important;
}
```

### Просмотр границ слоев:

```css
.blob {
  border: 2px solid red;
  filter: blur(0px);
}
```

## Известные проблемы

1. **Safari blur производительность** - на старых iPhone может быть небольшое подлагивание
   - Решение: уменьшить blur до 60px

2. **Цвета на разных экранах** - могут выглядеть по-разному
   - Решение: тестировать на целевых устройствах
